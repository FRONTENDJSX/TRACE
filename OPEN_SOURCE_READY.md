### 🚀 Quick Start

1. **Clone and setup**:
   ```bash
   cd TRACE
   python setup.py
   ```

2. **Configure environment**:
   ```bash
   cp env.example .env
   # Edit .env and add your GEMINI_API_KEY
   ```

3. **Run the demo**:
   ```bash
   python example.py
   ```

4. **Start API server**:
   ```bash
   python trace_api.py
   ```
